# Atmospheric, Climate, and Weather Tools

Software for automated data retrieval, processing, analysis, and visualization.

## Projects

The following projects were **designed, written, and tested entirely by a human author, without the use of AI coding agents or automated code-generation systems**:

- **hrrr-data**  
  Python toolkit for downloading and processing High-Resolution Rapid Refresh (HRRR) forecast data.

- **isd-lite-data**  
  Python toolkit for downloading and processing Integrated Surface Database Lite (ISD-Lite) observational data.
